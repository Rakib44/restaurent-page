# Royal Bangla Kitchen - Modern Restaurant Website

## 🍛 Project Overview
A beautifully crafted, fully responsive Bengali restaurant website showcasing authentic Bengali cuisine with a premium, minimal aesthetic. The design features warm Bengali heritage colors (red, gold, white) with smooth animations and modern functionality.

## ✨ Key Features Implemented

### 1. **Hero Section**
- Eye-catching headline: "Taste the Authentic Flavors of Bengal"
- Featured dishes with emojis (Kacchi Biryani, Hilsa Fish, Bhuna Khichuri)
- Prominent "Order Online" and "Reserve Table" CTAs
- Gradient text effects and smooth animations

### 2. **Navigation**
- Fixed top navigation bar with responsive hamburger menu
- Logo with restaurant name and icon
- Smooth scroll navigation to all sections
- Mobile-first responsive design

### 3. **About Section**
- Story of traditional Bengali cuisine
- Features list (homemade recipes, premium ingredients, expert chefs)
- Beautiful alpana pattern background with animations
- Two-column layout adapting to mobile

### 4. **Comprehensive Menu**
- 8 signature dishes across 4 categories:
  - **Biryani**: Kacchi & Pucka Biryani
  - **Fish**: Hilsa Macher Jhol, Bhuna Khichuri
  - **Meat**: Mutton Rezala, Chicken Kosha
  - **Desserts**: Rasgulla, Kheer
- Interactive category filtering
- Price display and "Add to Cart" buttons
- Smooth animations on hover

### 5. **Gallery Section**
- Visual showcase of 6 food & restaurant images
- Hover effects with scale transformations
- Responsive grid layout

### 6. **Customer Testimonials**
- 4 authentic customer reviews
- Star ratings in gold color
- Testimonial cards with left border accent

### 7. **Contact Section**
- Restaurant location, phone, and opening hours
- Google Maps embed
- WhatsApp ordering button (functional)
- Reserve a table button

### 8. **Interactive Features**
- **Shopping Cart Modal**: Add items, view totals, enter delivery details
- **Reservation Modal**: Book a table with date, time, and guest count
- Order notifications
- Smooth modal animations

### 9. **Footer**
- Quick links and social media icons
- Restaurant information
- Copyright details

## 🎨 Design Features

### Color Palette (Bengali Heritage Inspired)
- **Primary Red**: #D4501A (warm, appetizing)
- **Gold Accent**: #C4A747 (premium feel)
- **Dark Red**: #8B2D0B (contrast)
- **White**: #FFFFFF (clean background)
- **Light Background**: #FBF8F3 (warm cream)

### Typography
- **Headlines**: Playfair Display (elegant, serif)
- **Body**: Poppins (modern, clean, sans-serif)
- **Large headings**: Max 3.5rem
- **Proper hierarchy** throughout

### Animations & Effects
- Fade-in animations on page load
- Smooth hover effects on buttons and cards
- Scale transformations on gallery items
- Alpana pattern float animation
- Scroll-triggered animations on menu items
- Backdrop blur on modals

## 📱 Responsive Design
- **Mobile-first approach**
- Hamburger menu for tablets/mobile
- Adapted layouts for screens < 768px
- Touch-friendly buttons and interactive elements
- Optimized typography for small screens
